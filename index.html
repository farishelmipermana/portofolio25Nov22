link : tailwind
Portofolio pertama merupakan project yang dibuat menggunakan tailwind css. Ada fitur dark mode. Ketika scroll ke bawah, navbar tidak mengikuti layar. Logo abu abu jika di hover akan berwarna. Ada footer dan tombol untuk pergi ke halaman paling atas.

Ini adalah bootstrap versi 5. Terdapat foto yang dilingkari oleh border putih. Terdapat navbar yang mengikuti layar. Jika tombol navbar ditekan maka akan menuju destinasi. Terdapat animasi diam diantara section satu dengan section lainnya

Fifth

link : Express to MongoDB
Form diatas terintegrasi dengan NoSQL database yang bernama MongoDB. Nama hingga pesan yang ditulis akan terkirim ke dalam database MongoDB. Walaupun jika menekan tombol "save" akan terdapat error, tapi data terkirim kepada MongoDB.
